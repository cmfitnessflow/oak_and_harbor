# Harbor & Oak Outdoor Living Website

This folder contains a single-page luxury marketing website ready for GitHub and Cloudflare Pages deployment.

## Files

- `index.html` contains the full semantic page structure and Tailwind CDN setup.
- `style.css` contains custom styling, animations, and form states.
- `script.js` contains mobile menu behavior, smooth scrolling, and placeholder form handling.
- `assets/brand-mark.svg` is the local logo mark.
- `assets/favicon.svg` is the browser tab icon.
- `assets/site.webmanifest` provides basic install/display metadata.

## Deployment

Upload this folder to a GitHub repository and connect the repository to Cloudflare Pages. No build command is required.

## Customization

Update text directly in `index.html`. Replace image URLs in the hero, services, and about sections with final project photography when available.
